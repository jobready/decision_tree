# Decision Tree

Decision Tree is an easy way of defining rules/workflows

## Installation

Add this line to your application's Gemfile:

    gem 'decision_tree'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install decision_tree

## Usage

```ruby

class TestWorkflow < DecisionTree::Workflow
end

```
